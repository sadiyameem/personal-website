# Personal Website

# Description
This is a personal website about me showing my coding journey, hobbies, and achievements.

# Features
- Navigation bars
- Buttons
- Images
- Hover effect
- Question section
- Mobile user friendly
- Google fonts

# Tools
- HTML
- CSS

# Challenges/Learned
- At first some images weren't showing due to incorrect file names and arranging them in grids.
- I learned how to use flexbox to stack images and text neatly and use CSS grid to organize the layout.
